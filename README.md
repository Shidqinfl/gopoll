### GOPOLL ###

## Instruction ##
    - install go https://go.dev/doc/install on your device
    - open gopoll directory
    - go build . 
    - ./gopoll.exe in windows or ./gopoll in linux
    - ./gopoll --help to see the command
    - example command = ./gopoll -b 9600 -c 1 - i 1 -r 0x01 -p /dev/ttyUSB0
